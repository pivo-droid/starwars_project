# Star Wars Characters Project
Веб-приложение на Django для просмотра информации о 3 персонажах Star Wars из публичного API.

## Описание

Приложение позволяет:
- Просмотреть список персонажей Star Wars: Люк Скайуокер, С-3PO, Р2-Д2
- Получить детальную информацию о персонаже (рост, вес, цвет волос и т.д.)
- Увидеть список фильмов, в которых участвует персонаж
- Просмотреть транспорт и космические корабли персонажа

## Установка

1. Скачиваете файл из репозитория
2. Создайте виртуальное окружение
3. Проверьте версию Python, а также установите Django и Requests
4. Примените миграции: python manage.py migrate
5. Запустите сервер разработки: python manage.py runserver
6. Откройте браузер и перейдите на http://127.0.0.1:8000/ (можно из терминала)
